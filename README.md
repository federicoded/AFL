# Website Promocional - AFL Grafoanálisis
Sitio web promocional para presentar los servicios de AFL Grafoanálisis.
Construido con **React + TypeScript + Vite**.

## 💼 Servicios
Este sitio tiene como objetivo:
- Presentar los servicios ofrecidos
- Comunicar la propuesta de valor profesional
- Facilitar el contacto con potenciales clientes

## 🚀 Tecnologías utilizadas
- React
- TypeScript
- Vite
- HTML5
- CSS3
